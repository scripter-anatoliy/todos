# Todo-приложение

## Что нужно

- **Node.js** — лучше 18 или новее (проверь: `node -v`).

## Запуск

```bash
npm install
npm run dev
```

## Остальные команды

- `npm run build` — сборка в папку `dist` для выкладки.
- `npm run preview` — посмотреть собранный билд локально.
- `npm run deploy` — задеплоить на GitHub Pages.

## Стек

React 19, TypeScript, Vite, Redux Toolkit, SCSS. Задачи хранятся в localStorage.
